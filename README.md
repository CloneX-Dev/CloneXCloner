# CloneXCloner

# <img src="https://raw.githubusercontent.com/tandpfun/skill-icons/main/icons/Discord.svg" width="30"/> CloneX — Discord Server Cloner

**Профессиональный инструмент для полного клонирования структуры Discord-серверов**  
Быстро, безопасно и с детальным контролем процесса.

[![Discord](https://img.shields.io/discord/123456789012345678?label=Discord&logo=discord&color=543964)](https://discord.gg/fb2dMUpTHV)

---

## ✨ Возможности

- 🔄 **Полное клонирование структуры** — название, роли, категории, каналы, права доступа
- 🎯 **Точное копирование** — сохраняются позиции, цвета ролей, битрейт голосовых каналов и переопределения прав
- 🛑 **Мгновенная отмена** — остановите процесс в любой момент одним кликом
- 📜 **Детальные логи** — отслеживайте каждый шаг клонирования в реальном времени
- 🌓 **Тёмная / светлая тема** — комфортная работа в любых условиях
- 🔐 **Полная безопасность** — ваш токен хранится только в локальной сессии браузера и никуда не передаётся

---

## 🚀 Как использовать

1. **Войдите через Discord-токен** (инструкция по получению токена — [здесь](https://discord.com/developers/docs/intro))
2. **Введите ID исходного и целевого серверов** (или выберите из списка ваших серверов)
3. **Нажмите «Начать клонирование»** — и следите за прогрессом!

> ⚠️ **Важно:** перед клонированием **все каналы и роли на целевом сервере будут удалены**.  
> Используйте инструмент только на тех серверах, которыми вы владеете или имеете право администрирования.
